# SAP Fiori 原型设计工具功能清单

## 核心能力
- Fiori 风格原型画布，支持拖拽添加组件与快速点击添加。
- 组件可选中、移动、缩放，支持多选与批量样式设置。
- 属性面板实时编辑位置、尺寸、标签、值与样式。

## 画布与布局
- 画布尺寸可调（宽高可编辑）。
- 网格显示可开关，网格尺寸可调。
- 吸附（Snap）开关，便于对齐布局。
- 画布背景可切换：白色、SAP 浅蓝、黑色、浅蓝、蓝绿渐变。
- 居中视图按钮，自动滚动到画布中心。

## 组件库
- 基础组件：按钮、文本、图标。
- 输入组件：输入框、区间输入、下拉框、搜索框、多行文本、复选框、单选框、开关。
- 布局组件：卡片、表格、表单、分区、Tab、容器（GroupBox）、工具条。
- 流程组件：流程框、横幅、左右/上下箭头、矩形、菱形、终止符、直线、虚线、折线、折线箭头、虚线箭头。
- 组件库内带预览缩略图。

## 模板
- 视图选择模板（高保真 SAP 选择屏幕风格）。
- 接口示意图模板（系统接口流程图）。
- 模板支持一键应用，且可保存为自定义模板覆盖。

## 字段表格（Field Table）
- 自动从画布元素生成字段表格。
- 字段编码自动编号：001、002、003…
- 字段表格列：Field ID、Technical Field、Description、Field Type、ABAP Type、Length、Sample、Logic、Remarks。
- 表格可编辑（直接在表格单元格内修改）。
- 支持一键复制表格内容与下载 CSV。

## 交互与快捷键
- 双击元素可直接编辑标签。
- Cmd/Ctrl + Z 撤销。
- Cmd/Ctrl + Shift + Z 或 Cmd/Ctrl + Y 重做。
- Cmd/Ctrl + C 复制选中元素。
- Cmd/Ctrl + V 粘贴选中元素。
- Cmd/Ctrl + D 删除选中元素。
- Cmd/Ctrl + A 全选所有元素。
- Backspace/Delete 删除选中元素。

## 截屏与导出
- 一键截屏（优先复制到剪贴板，失败则下载 PNG）。
- 截屏时自动隐藏提示并进行字体加载优化。

## 多语言与样式
- 语言切换：English / 中文 / Français。
- 文本预设样式：标题、分区、标签、正文。
- 支持批量选择后统一设置字体大小、粗细、颜色。

## 持久化与记忆
- 画布元素、模板选择、语言、网格/吸附/编号状态持久化保存。
- 自定义模板覆盖持久化保存，刷新后可继续使用。
- 字段表格的编辑内容会持久保存。# SAP-DESIGNER
